# WhatsApp Bulk Sender Pro - Enterprise Edition

Complete bulk WhatsApp messaging solution with admin panel, reseller system, and API integration.

## Features

✅ **Admin Panel**
- User & Reseller Management
- Template Approval System
- Campaign Monitoring
- Analytics & Reports
- Credit Management
- Commission Tracking

✅ **Reseller System**
- Commission on Customer Sales
- Sub-user Management
- White Label Option
- Profit Tracking

✅ **User Features**
- Single & Bulk Message Sending
- Template Management
- Campaign Scheduling
- Message History
- Credit System
- API Integration
- Webhook Support

✅ **Technical Features**
- Official WhatsApp API Integration
- Personal Connection Support
- Proxy Support
- Rate Limiting
- Webhook Events
- JWT Authentication
- Database Encryption

## Requirements

- PHP 7.4+
- MySQL 5.7+
- Node.js 14+ (for personal connection feature)
- Apache/Nginx with .htaccess support
- OpenSSL extension
- cURL extension
